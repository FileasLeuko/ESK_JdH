Regressionen

Das Regressionsmodell: abhngige Variable ist MWh, regressiert auf die unabhngigen Variablen Temp, Sun, Wind und Deklination.

reg1 <- lm(data = Hemon, formula = MWh ~ Temp + Sun + Wind + Deklination)
