Regressionen

Das Regressionsmodell: abhängige Variable ist MWh, regressiert auf die unabhängigen Variablen Temp, Sun, Wind und Deklination.

reg1 <- lm(data = Hemon, formula = MWh ~ Temp + Sun + Wind + Deklination)

Analysen
